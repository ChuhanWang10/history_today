# History Today

History Today is a small web app that shows a random historical event from today's date and reads it aloud using ElevenLabs Text-to-Speech.

The browser fetches historical event data from Wikimedia. A small local Express server keeps your ElevenLabs API key private and exposes a `/api/tts` endpoint that returns MP3 audio to the frontend.

Check out the github pages: https://chuhanwang10.github.io/history_today/

## Install

```bash
npm install
```

## Configure

Create a local `.env` file:

```bash
cp .env.example .env
```

Add your ElevenLabs settings to `.env`:

```bash
ELEVENLABS_API_KEY=your_elevenlabs_api_key_here
ELEVENLABS_VOICE_ID=your_elevenlabs_voice_id_here
ELEVENLABS_MODEL_ID=eleven_multilingual_v2
PORT=3000
```

`ELEVENLABS_MODEL_ID` is optional and defaults to `eleven_multilingual_v2`.

Do not commit `.env`. It is ignored by git, and your ElevenLabs API key must never be committed.

## Run

```bash
npm start
```

Open:

```txt
http://localhost:3000
```

## Data

Historical events are loaded from Wikimedia's English "On this day" endpoint:

```txt
https://en.wikipedia.org/api/rest_v1/feed/onthisday/events/{MM}/{DD}
```

## Notes

The primary voice is generated by ElevenLabs TTS through the local backend. If server TTS fails, supported browsers can use the built-in browser voice as a fallback.
