# History today

History today is a small web app that shows a random historical event from today's date and reads it aloud using OpenAI Text-to-Speech.

The browser fetches historical event data from Wikimedia. A small local Express server keeps your OpenAI API key private and exposes a `/api/tts` endpoint that returns MP3 audio to the frontend.

## Install

```bash
npm install
```

## Configure

Create a local `.env` file:

```bash
cp .env.example .env
```

Add your OpenAI API key to `.env`:

```bash
OPENAI_API_KEY=your_openai_api_key_here
PORT=3000
```

Do not commit `.env`. It is ignored by git.

## Run

```bash
npm start
```

Open:

```txt
http://localhost:3000
```

## Data

Historical events are loaded from Wikimedia's English "On this day" endpoint:

```txt
https://en.wikipedia.org/api/rest_v1/feed/onthisday/events/{MM}/{DD}
```

## Notes

The primary voice is generated by OpenAI TTS through the local backend. If server TTS fails, supported browsers can use the built-in browser voice as a fallback.
